# Ferroptosis-Related Gene Analysis in Oral Submucous Fibrosis

## Project Overview
This project analyzes gene expression data from Oral Submucous Fibrosis (OSMF) tissues to identify ferroptosis-related genes using bioinformatics techniques.

## Dataset
Dataset used:
GSE64216 from NCBI GEO database.

## Methods Used
- Data preprocessing
- Differential Gene Expression (t-test)
- Ferroptosis gene filtering
- Volcano plot visualization

## Tools Used
Python
Pandas
NumPy
SciPy
Matplotlib

## Result
Identified ferroptosis-related gene:
ATG5
